Render - Ribbon

Overview

Drawing and parameters when "Ribbon" is selected in drawing setting are explained.

When "Ribbon" is selected, effects on continuous ribbon are drawn.

Ribbon is not drawn for each particle, it connects all children of a certain particle and draws it. For example, if there are particles 1, 2, and 3 of the child of particle 0, a shape like a ribbon drawn between 1, 2, and 3 is drawn. The ribbon is shaped like connecting the lines between (-0.5, 0, 0) and (0.5, 0, 0) of each particle.

Parameter

描画順

Specify the order in which particles are drawn among particles generated from the same node. In the order of generation, if the first generated particle is the first, in reverse order of generation, the first generated particle will be drawn last. That is, in the order of generation, the drawing of the last generated particle is given priority over the first generated particle.

視点追従

The shape of the ribbon is a line with a thickness connecting the lines connecting (- 0.5, 0, 0) and (0.5, 0, 0) points centered on the particle. Normally, the direction of the line connecting the two points depends on the rotation of the particle, but it specifies whether to turn it toward the view side or not by rotating it.

全体色

Specify the overall color of the ribbon.

頂点色

Specify the color of each vertex of the ribbon.

頂点座標

Specify the coordinates of each vertex of the ribbon. Only X coordinates can be specified here.
