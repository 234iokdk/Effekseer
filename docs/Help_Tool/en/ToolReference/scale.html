Scale

Overview

Set parameters related to node expansion.

Parameter

Scaling

Specifies the spawn scale of the particle. This will influence the scale of the particle relative to its parent particle. If there is no parent particle, or scale is set to not inherit in the Basic Settings window, this will specify the absolute scale instead.

Scaling Factor

Specify the enlargement ratio of the particle.

Scaling Factor, Expansion Speed, Expansion, Accel

Specify the enlargement rate, velocity, and acceleration of the particle.

Easing

Specify the enlargement ratio at particle generation and enlargement ratio at the end.

F-Curve

Use the F curve.
