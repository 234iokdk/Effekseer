Render - Sprite

Overview

Drawing and parameters when "Sprite" is selected in drawing setting are explained.

When "Sprite" is selected, squares, rectangles, arbitrary squares, etc. are drawn.

Parameter

Rendering Order

Specify from which sprite to draw in order from among sprites originating from the same node. In the case of generation order, the first generated is the last generated, the reverse of the generation order, the first generated is drawn before.

Configuration

Specify how to place sprites on particles. In the case of "Billboard", the sprite always points to the viewpoint side. In the case of "Z axis rotating building board", peel the viewing side while fixing the Z axis rotation. In the case of "Y axis fixed", peel the viewing side while fixing the rotation of the Y axis. When "fixed", the sprite follows the rotation of the particle.

Color All

Specify the color of the whole sprite.

Vertex Color

Specify the color of each vertex of the sprite.

Vertex Coords

Specify the coordinates of each vertex of the sprite.
