Sound

Overview

Set the parameters of the sound accompanying the node.

Parameter

Sound Data

Specify the WAV file of the sound to be played.

Volume

Specify the size of the sound to be played back. The size range is from 0 to 1.

  • 1：The size of the original sound
  • 0：Silence

Pitch

Specify the height of the sound to be played. Height range is -2 to 2.

  • 0：The height of the original sound
  • 1：Increase by 1 octave
  • -1：Decrease by 1 octave

Panning Type

This is the type for specifying balance of sound (left / right) balance. There are 2 types, 2D mode and 3D mode.

Pan

It is a parameter of 2D mode. The range is -1 to 1.

  • 0：Sound from the center
  • -1：Sound from the left
  • 1：Sound from the right

Damping Range

It is a parameter of 3D mode. It is natural that the sound is lost, the farther the distance is, the less heard it becomes. This is the distance value that begins to be smaller. The range is 0 ~.

Delay

It is the time from node creation to playback of sound. The range is 0 ~.
