No01.サンプルを見る＆使ってみよう！

概要

本章では、実際にEffekseerを起動してみてサンプルを見つつ基本的な操作方法を説明します。 その後、他のアプリケーション(ゲーム)で使用する方法等を説明します。

基本操作

起動

Effekseerは「Tool/Effekseer.exe」にあります。早速、ダブルクリックして起動してみましょう。 もしEffekseerが起動されない場合、.NetFramework4.0(Full)や最新のDirectXエンドユーザーランタイムがインストールされていない可能性があります。

  • Microsoft .NET Framework 4(Full)
  • 最新のDirectX エンド ユーザー ランタイム

それでも起動しない場合、公式サイトに問い合わせてみましょう。
